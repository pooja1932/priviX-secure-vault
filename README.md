# priviX - Secure Vault Protocol 🛡️

priviX is a sleek, highly-secured file management network offering military-grade encrypted storage. Built with an elegant app-like UI, priviX provides a seamless and visually stunning experience for managing and sharing sensitive assets across the network.

## 🌟 Key Features
* **Modern Interface**: Full integration of glassmorphism design with responsive, off-canvas navigation.
* **Animated Aesthetics**: Fluid particle-network background, animated orbs, and dynamic CSS styling.
* **File Management System**: Instantly upload, search, manage, and delete files in your secure personal vault. 
* **Asset Sharing Protocol**: Securely grant or revoke access to uploaded files with other participants in the network.
* **Secure Authentication**: Encrypted credentials with beautiful, simple access portals.

## 🛠️ Stack
* **Python** (Flask)
* **SQLite3**
* **HTML5 / CSS3 / JavaScript** 
* **Bootstrap 5** & **FontAwesome**

## 🚀 Getting Started
1. Clone the repository.
2. Ensure you have Python installed.
3. Start the local server:
    ```bash
    python app.py
    ```
4. Access the web interface at `http://127.0.0.1:5000`
